### 分析词义

"Vite"（法语 vite，意为“快”）由 Vue 作者 Evan You 推出，支持 Vue、React、Svelte 等。dev server 按需编译模块，HMR 极快；`vite build` 输出优化后的静态资源。Astro 等框架也采用 Vite 作为底层。

### 列举例句

1. "Create a new project with npm create vite@latest." (用 npm create vite@latest 创建新项目。)
2. "Vite pre-bundles dependencies with esbuild for speed." (Vite 用 esbuild 预打包依赖以提速。)
3. "The Astro site uses Vite under the hood for builds." (该 Astro 站点底层用 Vite 构建。)

### 词根分析

法语 vite（快速），品牌刻意强调开发体验速度。

### 词缀分析

外来词品牌名，无英语词缀。

### 文化背景

2020 年后大量项目从 Webpack 迁移到 Vite；与 Vitest 测试框架同源生态。

### 单词变形

- 固定搭配: "Vite dev server" (Vite 开发服务器), "Vite plugin" (Vite 插件), "Vite config" (Vite 配置)
- 相关: Rollup, esbuild, HMR

### 记忆辅助

法语 vite = 快——Vite 就是追求快的构建工具。

### 助记故事

"Switching to Vite cut local dev reloads from seconds to milliseconds." (改用 Vite 后，本地开发热更新从秒级降到毫秒级。)
