### 分析词义

"Requested" 说明某物因对方要求而提供或执行，如 requested documents（按索要的文件）、as requested（照要求）。比 asked 更正式，适合客户、上级或官方场合。

<Pronunciation word="Requested" uk="[rɪˈkwestɪd]" us="[rɪˈkwestɪd]" client:only />

### 列举例句

1. "Please find the requested files attached." (请查收所索要的附件文件。)
2. "The manager requested a full report by Friday." (经理要求周五前提交完整报告。)
3. "Access was granted only to requested users." (仅向经申请的用户开放访问。)

### 词根分析

"Request" 来自古法语 requeste，源于拉丁语 requirere（再次寻求），与 require 同源。

### 词缀分析

re-（再次）+ quest（寻求）→ request + -ed；名词 request 与动词同形。

### 文化背景

邮件主题常写 "Re: requested information"；客服 "fulfill a requested refund"；法律 "requested discovery" 指按程序要求的证据开示。

### 单词变形

- 固定搭配: "as requested" (照要求), "requested information" (所要求的信息), "upon request" (应要求)

### 记忆辅助

request 是“再求一次”，requested 是“已经被求到/按求提供的”。

### 助记故事

"She sent the requested invoice the same afternoon." (她当天下午就寄出了对方索要的发票。)
